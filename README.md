# Recipe Website

A clean and modern recipe website designed to showcase recipes, ingredients, preparation steps, and cooking tips. This project aims to provide users with an intuitive interface for discovering and following recipes.

## Features

- Recipe listing with categories
- Detailed recipe pages with ingredients and instructions
- Searchable and easy-to-navigate layout
- Responsive design for desktop and mobile devices

## Project Structure

- `index.html` - Main landing page
- `recipes.html` - Recipe listings and categories
- `recipe-detail.html` - Individual recipe details
- `styles.css` - Global styles and layout
- `script.js` - Interactive page behavior

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/recipe-website.git
   ```
2. Navigate to the project folder:
   ```bash
   cd recipe-website
   ```
3. Open `index.html` in your browser or use a local development server.

## Usage

- Browse recipes from the homepage.
- Click any recipe card to open the full recipe details.
- Follow the ingredient list and preparation steps to recreate the dish.
